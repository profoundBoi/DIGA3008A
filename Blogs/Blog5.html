Blog 7: possible website analyse.

1. Gandi.net

This website a website that offers a service where they host people's domain registrations.

Pros of the Gandi.net website:

Pros of this analysis is that the website is setup clearly which will make analysing each section of the UI and UX simpler to approach.

Cons of the Gandi.net website:

Con if I choose to do this analyse is that they also create websites for their users which might make it a little more difficult to analyse as I am still a beginner at making websites where they have been doing it for years. Their experience would have helped them create a website that's difficult to find mistakes on if that was what I wanted to critique.

2. Supersport.com:

The super sport website offers it users the most popular streaming for sports, news, clips and other media-based entertainment and statistics.

Pros of the Super sport website:

  • The website keeps it navigation consistent so that we do not get confused in terms where we are on the website.
  • It has a red line to below the navigation bar that the user is on, so we don't get lost.
  • It has hover effects for anything interactable that the user hovers over so that they know this part of the website is interactable.
  • Some pages have a load more button if the users decides that they would like to continue browsing. It does this so that the user isn't presented with a lot of information at one time which can become overstimulating.

    • Cons of the Super Sport website:

  • When the user clicks on some tabs it automatically takes them to a new website, for example the news tabs. I feel like this distracts the user from their goal on the website. Instead, they should just update the news regularly and reference the sauce.
